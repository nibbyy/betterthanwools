# betterthanwools
BetterThanWools is a 26.1.2 implementation of the wool processing and Knitting system from BetterThanWolves.

**BetterThanWools brings the wool processing line and Knitting system from [Better Than Wolves](https://wiki.btwce.com/view/Main_Page) to modern Minecraft.**

🛠️ **Mechanics:** Sheep produce Raw Wool instead of complete wool blocks. Process it by hand with Knitting Needles, then use the finished Wool Knits to make early-game armor, bedrolls, and other colorful wool products.

⚙️ **Config:** Through Cloth Config and Mod Menu, BetterThanWools is highly configurable. In its default state the mod is configured to mirror the BetterThanWolves wool mechanics as close as possible.

*Features marked as **New** are 'new' features not included in the original Better Than Wolves mod.*

## 🧾 Requirements

- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Slowcraft](https://modrinth.com/mod/slowcraft) library
- (Optional) [JEI](https://modrinth.com/mod/jei)
- (Optional) [Mod Menu](https://modrinth.com/mod/modmenu) and [Cloth Config](https://modrinth.com/mod/cloth-config)

## 🪡 Knitting

Knitting Needles can be crafted with two sticks. Combine two Raw Wool with Knitting Needles to create a Knitting tool.

Hold right-click with the Knitting to work on it while its durability bar shows your progress. Once completed, it produces a Wool Knit and returns the Knitting Needles.

Knitting progress is saved whenever you stop, allowing Knits to be completed a little at a time.

**New:** The default knitting time is 60 seconds and can be changed in the configuration.

## 🐑 Wool Processing

BetterThanWools adds a more involved progression between sheep and finished wool products:

- Sheep drop and produce Raw Wool instead of complete wool blocks
- Raw Wool can be dyed and combined into all 16 wool colors
- Knitting turns Raw Wool into finished Wool Knits
- Wool Knits are used to craft armor and bedrolls
